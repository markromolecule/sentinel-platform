# Goal

- To create a effective yet clean and simple dashboard / overview for the support dashboard 
- I want you to fully refactor the current layout of the support dashboard to be a more effective overview of the support function of the organization. 
- It should help the user to identify on the datas that's coming from the system end-to-end
- In terms of styling, I require it to be simple, clean and effective in terms of the data that you will be inserting to show the user on what's going on under the system. In addition, you should follow the margins I used across the other pages
- I want a dashboard that was customizable to a layout;  the user can move the cards, metrics, data visualizations to prioritize on what the user wants.
- If there are multiple metrics that require a graph, group it in a way that is most effective to look at. In addition for cards, you should implement a way where the user can drag it right to left like a carousel but instead of click- the user can drag it horizontally to view other metrics.
- Prioritize the data that's needed to show on the overview, what's essential and what needs to be seen by the user. I want to be able to see everything in one go
