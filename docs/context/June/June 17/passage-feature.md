# Goal

This document outlines the enhancement of the passage feature within the Sentinel platform, specifically focusing on its application to questions during examination attempts. The primary objective is to introduce greater flexibility and richness to the content displayed alongside questions, moving beyond the current limitations.

Currently, the system only supports a basic `[reference]` field for questions, which is insufficient for providing comprehensive context or rich media to students during an examination. This often leads to a suboptimal examination experience, as instructors are unable to fully articulate complex scenarios or provide necessary supplementary materials directly within the question context.

The improved passage feature aims to address these shortcomings by allowing instructors to override the default plain passage with custom, rich content. This will be facilitated within the question [builder] interface, enabling instructors to:

1.  **Embed Images**: Instructors can include relevant images to illustrate concepts, diagrams, or visual data, enhancing clarity and understanding for students.
2.  **Integrate Code Snippets**: For technical subjects, instructors can embed code snippets. The system will be enhanced to project these snippets with proper syntax highlighting and formatting, mimicking an actual code editor environment for better readability and comprehension.
3.  **Add Short Descriptions**: Beyond just images and code, instructors can provide concise, formatted descriptions that offer additional context, background information, or specific instructions pertinent to the question.

This enhancement will significantly improve the user experience (UX) for both instructors and students across all [question-types]. For instructors, it provides powerful tools to create more engaging and informative assessments. For students, it ensures that all necessary information is presented clearly and accessibly, directly within the examination interface, leading to a more standardized and effective learning assessment. The ultimate goal is to elevate the overall examination experience by making passages a dynamic and integral part of question presentation.
