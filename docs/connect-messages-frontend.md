# Goal

- To connect the backend [messages] module to frontend
- Check the mutations @packages/hooks/src/query/messages and @packages/services/src/api/messages.ts
- Ensure that the realtime connection is working
- Ensure that the user will get the correct response and will be able to send messages and the messages will be displayed in the UI. Moreover, they can communicate all the users under the system. [Including] other institutions, parent institutions, branches.
- Ensure that each [profile] of the user has a corresponding [institution] indicated to their profiles
- Ensure that the user is already authenticated before accessing the messages module
- Ensure that each user has [active] or [status] included on the frontend and also check the backend.
