# Issue

- Bulk-import is showing on the admin page for the department page

1. You should create a permission that will allow and deny the action of bulk-import on the department page

2. The superadmin and admin should only view the departments under their parent institution and specific branch institution this also applies on the courses, semesters, sections, etc.

3. Notification only shows create notifications. For example, departments; it only shows departement created and does not show if the data is updated, deleted, etc. This should also apply to other modules across the system.
